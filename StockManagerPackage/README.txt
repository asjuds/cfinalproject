========================================
  Stock Management System
  Portable Distribution Package
========================================

INSTALLATION:
-------------
1. Extract this entire folder to any location on your computer
   (e.g., C:\Program Files\StockManager or Desktop)

2. Double-click "RunStockManager.bat" to start the application

3. No installation required! All necessary files are included.

USAGE:
------
- Add Product: Register new products with ID, name, category, price, and quantity
- Update Stock: Increase stock levels for existing products
- Sell: Record sales and automatically update stock
- Check Stock: View current stock levels with low stock warnings
- Calculate Value: See total value of all inventory
- Apply Discount: Calculate discounted prices (10-20% discount)
- Remove Product: Delete products from the system
- Generate Report: View detailed reports of all operations

DATA STORAGE:
-------------
All data is automatically saved in the "data" folder:
- products.csv: All product information
- history.csv: Complete operation history

You can backup these files to preserve your data.

SYSTEM REQUIREMENTS:
--------------------
- Windows 10 or later
- No additional software installation needed
- All required libraries are included in this package

TROUBLESHOOTING:
----------------
If the application doesn't start:
1. Make sure you extracted ALL files from the zip
2. Try running "stock_manager.exe" directly
3. Check that all DLL files are in the same folder as stock_manager.exe

For support or questions, contact your system administrator.
