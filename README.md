# Stock Management System

A complete GUI-based stock management application built with C and GTK4.

## Features

- Product management (add, update, remove)
- Stock updates and sales tracking
- Low stock warnings (quantity < 5 highlighted in red)
- Total inventory value calculation
- Discount system (10-20%)
- Complete operation history
- CSV data persistence
- Sortable product tables
- Modern GTK4 interface

## Installation

### For End Users

1. Download `StockManager_Installer.exe`
2. Double-click to install
3. Run from Start Menu or Desktop shortcut
